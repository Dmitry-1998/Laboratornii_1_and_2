Лабораторная работа 1

Данный пакет pасчитывает факториал чисел от 0 до 20.
К данной лабораторной относится папка laba_1, в папке src.
Типом переменной для сообщения от издателя std_msgs/Int64.
Данный тип переменной может принимать значения от 
-9223372036854775808 до 9223372036854775808.
Факториал числа 21 является 51090942171709440000, что уже не
помещается в данный тип данных.
Таблица факториалов: https://tab.wikimassa.org/tablitsa_faktorialov_do_50
